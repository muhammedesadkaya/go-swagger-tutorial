# Golang Swagger Implementation

This project has been prepared for the purpose of example of Swagger and JWT integration with Golang v1.18

## Libraries

1. gin-gonic/gin
2. swag
3. jwt-go

## Installing

```
make run (if you has make) or go run main.go
```
